# Retention, Habit Formation & Growth Dynamics in a Freemium Model

## Business Case: Dognition User Engagement Analysis  
**Author:** Ashma Poudel   
**Tools Used:** Tableau, Excel, GitHub  

---

## 🔍 Project Overview

This project analyzes user engagement behavior within Dognition’s freemium model to uncover retention gaps, habit formation patterns, and revenue growth volatility.

Using 177,000+ user test records, the analysis identifies where engagement drops, what drives long-term completion, and how monetization timing impacts user behavior.

The objective:  
Transform behavioral insights into actionable business recommendations.

## 🧠 Key Business Questions

- Where do free users drop off?
- Does engagement consistency predict completion?
- What time of day drives the highest activity?
- Is growth sustainable or promotion-driven?
- How can retention be improved before monetization?

---

## 📈 Key Insights

### 1️⃣ Retention Cliff at Tests 7–10  
Free users experience a significant drop-off before conversion stage.

### 2️⃣ Consistency Drives Completion  
Shorter gaps between tests strongly correlate with higher completion rates.

### 3️⃣ Evening Peak Usage (6–8 PM)  
User activity is concentrated post-work hours, suggesting behavioral positioning opportunities.

### 4️⃣ Growth is Promotion-Driven  
Revenue spikes align with free access promotions, indicating unstable organic growth.

---

## 💡 Strategic Recommendations

### Retention Optimization
- Extend free test access to increase habit formation
- Introduce streak-based rewards
- Trigger upgrade prompts during peak engagement hours

### Behavioral Reinforcement
- 24-hour reminder nudges
- Progress continuation prompts
- Consistency badge system

### Revenue Stabilization
- Reduce dependency on promotional spikes
- Introduce mid-tier pricing model
- Optimize referral triggers during high activity windows

---

## 📊 KPIs to Track

- Average tests completed per user
- Retention rate between Tests 7–10
- Average engagement gap
- Free-to-Paid conversion rate
- Monthly recurring revenue stability
